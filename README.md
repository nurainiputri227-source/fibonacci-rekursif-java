🧾 Deskripsi

    Program ini merupakan implementasi deret Fibonacci menggunakan bahasa Java dengan pendekatan rekursif.
    Pengguna diminta untuk memasukkan jumlah suku (n), kemudian program akan menampilkan deret Fibonacci hingga suku ke-n.

⚙️ Cara Kerja

    - Program menerima input jumlah suku Fibonacci dari pengguna
    - Menggunakan fungsi rekursif fib(n) untuk menghitung nilai Fibonacci
    - Jika n ≤ 1, maka nilai langsung dikembalikan (base case)
    - Jika n > 1, maka dihitung dengan rumus:
            fib(n) = fib(n-1) + fib(n-2)
    - Hasil ditampilkan dalam bentuk deret

🧠 Konsep yang Digunakan

    - Rekursi (Recursive Function)
    - Percabangan (If-Else)
    - Perulangan (Looping)
    - Input pengguna dengan Scanner

💻 Contoh Output

    Masukkan jumlah suku Fibonacci: 7
    Deret Fibonacci hingga suku ke-7 adalah:
    0, 1, 1, 2, 3, 5, 8
